## 📝 Assignments & Practice

### 📁 A1 – HTML Basics

1. **ElementsQ1.html**  
   Demonstrates the usage of commonly used HTML elements and tags, including:
   - Headings
   - Paragraphs
   - Lists
   - Links
   - Basic semantic structure

   This assignment focuses on understanding the core building blocks of HTML webpages.

2. **NestedElementsQ2.html**  
   Implements nested HTML elements and lists:
   - Ordered Lists
   - Unordered Lists
   - Multi-level Nested Lists
   - Structured Content Layout

   This exercise improves understanding of content hierarchy and webpage organization.

### 📁 A2 – Mini Frontend Project

1. **Cat.html**  
   A themed mini webpage project designed using:
   - Structured HTML layout
   - Semantic elements
   - Organized webpage sections

   The project focuses on combining HTML structure with CSS styling to create a visually appealing webpage.

2. **style.css**  
   Contains the CSS styling rules used for:
   - Webpage appearance
   - Layout customization
   - Typography styling
   - Colors and spacing
   - Overall design enhancement

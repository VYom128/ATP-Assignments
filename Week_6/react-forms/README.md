# React Forms

This project demonstrates how to effectively manage and validate forms in React using `react-hook-form`. It covers controlled versus uncontrolled components, form state management, and real-time validation.

## 🚀 Features
- **React Hook Form**: Utilizes `react-hook-form` for performant, flexible, and extensible forms with easy-to-use validation.
- **Form Validation**: Client-side validation for required fields, minimum lengths, and pattern matching, along with real-time error message rendering.
- **Dynamic User List**: Upon successful form submission, the new user's data is dynamically rendered onto the page.
- **Tailwind CSS**: Styled using Tailwind utility classes for a clean and responsive user interface.

## 🛠 Tech Stack
- React (Vite)
- `react-hook-form`
- Tailwind CSS

## 📁 Project Structure
- `src/components/FormDemo.jsx`: Basic form implementation and syntax.
- `src/components/FormAssignment.jsx`: A complete assignment demonstrating input capturing, complex validation rules, and displaying the submitted state in a formatted UI.
- `src/components/TestedTypes.jsx`: Demonstration of different input types and their specific validations.

## ⚙️ Installation & Setup
1. Navigate into the project directory: `cd react-forms`
2. Install dependencies: `npm install`
3. Start the Vite development server: `npm run dev`
